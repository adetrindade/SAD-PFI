# SAD-PFI
SAD-PFI | Sistema de Apoio à Decisão para Planejamento Financeiro Institucional
---
O SAD-PFI é um sistema computacional desenvolvido para apoiar o processamento, análise e geração de previsões a partir de séries temporais associadas à execução orçamentária de instituições públicas.
O sistema foi concebido como uma ferramenta de apoio à tomada de decisão, permitindo que gestores e pesquisadores analisem dados históricos e realizem projeções futuras por meio de modelos de previsão estatísticos e computacionais.
---
# Instalação e Execução

Para utilizar a versão mais recente, siga os passos abaixo:

1. Acesso a seção [**Releases**](https://github.com/adetrindade/SAD-PFI/releases/download/beta/SAD-PFI-v0.9-beta.1.zip) deste repositório.
2. Baixe o arquivo compactado mais recente.
3. Extraia o conteúdo em uma pasta de sua preferencia.
4. Instalar o .Net Runtime mais recente (No Windows 11 ao abrir o sistema caso não esteja instalado será solicitado a instalação automaticamente).
5. Instalar o Python 3.12 ou superior.
6. Executar o arquivo principal da aplicação "sadpfi.exe".
